# Overview

- Button コンポーネントは、ボタンのスタイルを提供します。
- Props を使用して、ボタンのスタイル等をカスタマイズできます。

## ボタンの種類

- variant を指定することで、ボタンのスタイルを変更できます。

### Primary

- 基本のスタイル

<Canvas of={ButtonStories.Primary} />

```tsx
<Button>Button</Button>
```

### Secondary

- セカンダリーのスタイル（白抜き）

<Canvas of={ButtonStories.Secondary} />

```tsx
<Button variant="secondary">Button</Button>
```

### Text

- テキストボタン（背景色なし）

<Canvas of={ButtonStories.Text} />

```tsx
<Button variant="text">Button</Button>
```

### Cancel

- キャンセルボタン

<Canvas of={ButtonStories.Cancel} />

```tsx
<Button variant="cancel">Button</Button>
```

### Danger

- 削除ボタン

<Canvas of={ButtonStories.Danger} />

```tsx
<Button variant="danger">Button</Button>
```

## ボタンのサイズ

- size を指定することで、ボタンのサイズを変更できます。

### Small

- 小さいサイズ

<Canvas of={ButtonStories.Small} />

```tsx
<Button size="sm">Button</Button>
```

### Medium

- 標準のサイズ

<Canvas of={ButtonStories.Medium} />

```tsx
<Button size="md">Button</Button>
```

### Large

- 大きいサイズ

<Canvas of={ButtonStories.Large} />

```tsx
<Button size="lg">Button</Button>
```
